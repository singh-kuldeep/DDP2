Function \hyperlink{InitialCondition_8h_a0f12b364366f7f909837cd56e0cb3f28}{initial\+\_\+condition()} defines the initial values of the conserved quantities in the live cells. 

There are \char`\"{}5\char`\"{} options are available for initial condition. Which are


\begin{DoxyEnumerate}
\item Initial\+Condition = Zero\+Velocity Zero velocity inside the domain and total quantities are given at the inlet
\item Initial\+Condition = Free\+Stream\+Parameter\+Everywhare Free stream parameters are filled inside the domain
\item Initial\+Condition = Free\+Stream\+Parameter\+And\+Zero\+Velocity
\item Initial\+Condition = Start\+From\+Previous\+Solution
\item Initial\+Condition = Nozzle\+Based\+On1\+D\+Calculation
\end{DoxyEnumerate}

Parameters are varying inside the nozzle domain based on the 1D is-\/entropic relation.

\begin{DoxyWarning}{Warning}
\char`\"{}\+Initial\+Condition = Nozzle\+Based\+On1\+D\+Calculation\char`\"{} Is only valid for nozzle simulation. DO N\+OT use it for any other simulation
\end{DoxyWarning}
